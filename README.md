# Исторические даты
В проекте использовал 
  - Swiper
  - Gasp
  - TypeScript
  - React

# Для запуска проекта

# Клонирование репозитория
```sh
git clone https://github.com/EmilMustafin/historical-dates.git
```
# Установка зависимостей
```sh
npm install
```

# Запуск проекта
```sh
npm run start
```
